# MrBuilderNYC — export scaffold

Standalone Vite project used when exporting a Nexora client on the **MrBuilderNYC** template.

- **Live admin overlay:** `src/template-mrbuildernyc/` (components, contexts, data, pages, lib)
- **Registry:** `id: "mrbuildernyc"` in `src/lib/templates.ts` and `server/export-logic.mjs`
- **Preview:** `/preview-mrbuildernyc.html` → `src/template-mrbuildernyc/main.tsx`
- **Showcase:** `/templates/mrbuildernyc`

This folder is synced from the live template under `src/template-mrbuildernyc` so exports and the repo layout stay aligned.
